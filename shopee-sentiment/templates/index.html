Apa Itu Naive Bayes?

Naive Bayes merupakan algoritma klasifikasi yang menggunakan Teorema Bayes untuk menentukan probabilitas suatu data termasuk ke dalam kelas tertentu. Algoritma ini terkenal cepat, ringan, dan sangat efektif untuk analisis sentimen teks.

Teorema Bayes

Menghitung probabilitas suatu kelas berdasarkan fitur-fitur yang dimiliki data.

Cepat Diproses

Cocok digunakan untuk ribuan hingga jutaan data ulasan dengan waktu pelatihan singkat.

Akurat

Sangat populer untuk klasifikasi teks seperti spam detection dan analisis sentimen.

Machine Learning

Model belajar dari data latih untuk memprediksi sentimen ulasan baru.

Rumus Teorema Bayes

P(C | X) = P(X | C) × P(C) P(X)
P(C|X) – Probabilitas sentimen C diberikan teks X (posterior)
P(X|C) – Likelihood: probabilitas teks X jika kelas adalah C
P(C) – Prior: probabilitas kelas C dalam data latih
P(X) – Evidence: konstanta normalisasi (diabaikan saat prediksi)
